#GIT
##常用命令
    git init                            #初始化一个空的git目录
    git remote add origin <url>         #添加远程仓库地址
    git pull                            #拉取远程更新
    git add <file>                      #添加到版本管理
    git commit -m "<message>"           #提交修改
    git push origin <branchName>        #上传代码
